这一课讲什么

veRL 学习实验室 ​

一套原创、渐进式的中文课程：从 LLM/RL 直觉与必要数学开始，完成第一次可验证训练，再沿单条样本阅读 veRL V1 源码，最后进入自定义算法和性能优化。

课程绑定 veRL 提交 e5687fce0516d31e1fdc4580499074a9bd94c751，默认讲解 trainer.use_v1=true；legacy V0 仅用于解释旧资料的迁移边界。

本地运行 ​

bash
cd verl-learning
npm install
npm run dev

生产校验：

bash
npm run check:diagrams
npm run build
npm run preview

部署到子路径：

bash
BASE_PATH=/verl-learning/ npm run build

静态产物位于 .vitepress/dist/。源码升级后，优先重查 main_ppo.py、ppo/v1/trainer_base.py、ppo/v1/agent_loop_tq.py、ppo/v1/replay_buffer.py、core_algos.py 与组合配置，然后更新课程的版本边界和源码链接。

CI 可运行 npm ci && npm run check:diagrams && npm run build。开发服务器默认监听本机回环地址；对外提供时发布静态产物。
