# losh

Personal blog for AI infrastructure and life.

## 写新文章

在 `src/content/blog/` 新建一个 `.md` 文件，使用下面的开头：

```md
---
title: '文章标题'
description: '一句摘要'
pubDate: '2026-07-14'
tags: ['AI', '学习']
---

正文从这里开始。
```

提交到 `main` 后，GitHub Pages 会自动更新网站。

## Pages CMS 编辑

打开 [Pages CMS](https://app.pagescms.org)，用 GitHub 登录并为 `Cass998/losh_has_blog` 安装 Pages CMS GitHub App。之后进入仓库的「博客文章」即可在手机或电脑上写作、上传图片；保存会直接提交到 `main` 并自动发布。
